# 🌐 Personal Portfolio Website

This is my personal portfolio website built using **pure HTML and CSS**.  
It showcases my education, skills, projects, experience, and contact details in a clean and structured layout.
---
## 🚀 Features
- ✅ Fixed Navigation Bar
- ✅ Smooth section-based layout
- ✅ Education details table
- ✅ Skills displayed using card design
- ✅ Projects section with detailed descriptions
- ✅ Experience section
- ✅ Contact section
- ✅ Resume download option
- ✅ Hover effects and simple animations
- ✅ Fully built using HTML & CSS (No frameworks)
---
## 🛠️ Technologies Used
- HTML5
- CSS3
- Flexbox
- CSS Variables
- Basic Animations & Hover Effects
